# Ansible Collection - octupus.o4n_automation_tools

## Octupus Collection

Collection o4n_automation_tools helps developers to develop automation Ansible software.  
By Ed Scrimaglia

## Required

Ansible >= 2.10

## Modules

o4n_unavailable_hosts
The module finds either using a static or dynamic inventory what hosts have connecion issues. If needed, the module create a report
with those hosts.
